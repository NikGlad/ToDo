Когда программа предлагает введите команду:, надо ввести одну из команд: help, add, show, random.
Когда программа предлагает введите дату для добавления задачи: надо ввести дату(числами или буквами)
Когда программа предлагает ввести название задачи: надо ввести назваение задачи.
Для отображения на экране введённых ранее задач надо набрать команду show, далее ввести дату задач для отображения.

Действия, которые будут выполняться при выборе одной из команд:
help - напечатать справку по программе.
add - добавить задачу в список (название задачи запрашиваем у пользователя).
show - напечатать все добавленные задачи.
random - добавлять случайную задачу на дату Сегодня
